# Blueprint: V-World 기반 토지 정보 조회 애플리케이션

## 1. 개요

본 프로젝트는 V-World API를 활용하여 사용자가 지도와 상호작용하며 특정 필지의 토지 정보를 조회, 관리하고, 그 결과를 CSV 파일로 내보낼 수 있는 웹 애플리케이션입니다. 모바일과 데스크톱 환경 모두에 최적화된 반응형 UI를 제공하며, 직관적인 시각화와 데이터 관리를 목표로 합니다.

## 2. 기존 주요 기능

*   **지도 엔진**: V-World 2.0 (2D OpenLayers 기반) 및 3.0 (3D WebGL 기반) 동시 지원.
*   **지적 정보 조회**: 지도 클릭 시 WFS(`lt_c_landinfobasemap`)를 활용하여 필지 경계 및 기본 속성(PNU, 주소, 지목, 면적 등) 조회.
*   **필지 하이라이트**: 선택한 필지를 2D 및 3D 지도상에 빨간색 경계선과 반투명 채우기로 강조.
*   **목록 관리**: 조회된 토지 정보를 카드 형태로 리스트업하고, 면적 합계 및 공시지가 총액 자동 계산.
*   **데이터 내보내기**: 수집된 토지 목록을 CSV 파일로 저장.
*   **주제도 제어**: 연속지적도, 용도지역, 지구단위계획 등 다양한 레이어 온/오프 및 순서 변경.
*   **환경성평가 연동**: ECVAM API를 통해 국토환경성평가 등급 및 생태자연도 등급 비동기 조회.
## 3. 완료된 기능 개선 (주제도 속성 조회 및 PWA 안정화)

사용자의 요청에 따라 **지구단위계획구역명** 상세화, **모바일 PWA 접속 안정성**, 그리고 **모바일 UI 편의성**을 대폭 강화하였습니다.

1.  **모바일 UI 레이아웃 최적화 (2026-03-03)**:
    *   **주요 버튼 위치 조정**: 모바일 환경에서 [토지조서 내보내기], [토지목록보기], [전체 삭제] 버튼이 검색창 바로 아래(지도 직후)에 위치하도록 레이아웃 순서(`order`)를 변경하였습니다. 이를 통해 사용자가 패널을 열자마자 핵심 기능을 즉시 사용할 수 있도록 개선하였습니다.
    *   **컴포넌트 재배치**: 상태 메시지 및 카운트 바의 순서를 조정하여 정보 가독성을 높였습니다.

2.  **모바일 PWA 안정화 (접속 오류 해결)**:
... (기존 내용 유지)

    *   **캐시 전략 고도화**: `sw.js`를 v2로 업데이트하여 기존 캐시 자동 삭제(Activate), 네트워크 장애 시 캐시된 `index.html` 서빙(Fallback) 로직을 구현하였습니다. 이를 통해 모바일 홈 화면에서 앱 실행 시 "사이트에 연결할 수 없음" 오류가 발생하는 문제를 해결하였습니다.

## 4. 향후 계획

*   **통계 기능**: 수집된 필지들의 용도지역별 면적 비율 등을 차트로 시각화.
*   **검색 필터**: 수집된 목록 내에서 특정 지목이나 면적 범위로 필터링하는 기능 추가.
*   **사용자 설정 저장**: 브라우저 로컬 스토리지를 활용하여 사용자가 추가한 필지 목록을 새로고침 후에도 유지.
